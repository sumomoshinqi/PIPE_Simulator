本目录下包括PIPE Simulator的全部内容

- bin文件夹内是编译好的可执行程序，主程序为 PIPE Simulator.exe

- doc文件夹内是软件的手册，包括软件的开发环境、运行环境、程序功能、使用说明、实现细节

- src文件夹内是源代码，其中PIPE Core.cpp是内核的实现文件，PIPE Simulator文件夹内是工程文件

- TestFiles文件夹内包含测试用的.yo文件

- Keynote文件夹内为展示所用幻灯片
